# Prime Number Finder

This Java program finds and prints prime numbers between 2 and 100.

## How to Use

1. Ensure you have Java installed on your system.
2. Compile and run the program.
3. The program will display prime numbers between 2 and 100.

## Example Output

2 is a prime number.
3 is a prime number.
5 is a prime number.
7 is a prime number
